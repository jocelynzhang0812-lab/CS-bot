# kb_06 — 会员权益

## 一、会员与额度

### 1. 会员套餐有哪些？

| 套餐 | 价格 | Kimi Claw | 额度 |
|------|------|-----------|------|
| Adagio | ¥0/月 | ❌ | 基础额度 |
| Andante | ¥49/月 | ❌ | 30 个 Agent |
| Moderato | ¥99/月 | ❌ | 60 个 Agent |
| Allegretto | ¥199/月 | ✅ | 150 个 Agent |
| Allegro | ¥699/月 | ✅ | 360 个 Agent |

> **注意**：Kimi Claw 功能仅 Allegretto 及以上套餐支持。

### 2. 如何查看额度？

- **Web 端**：左下角头像 → 设置 → 订阅
- **APP 端**：我的 → 会员计划 → 订阅

### 3. 额度用完了怎么办？

- 等待自动刷新（5 小时/周度/月度）
- 升级至更高等级会员
- 参与官方活动获取赠送额度

### 4. API rate limit 报错是什么意思？

表示 API 调用频率或额度已达上限。请检查：
1. 会员额度是否充足
2. 是否触发频次限制（5 小时及周频控）

详细查看：https://www.kimi.com/code/console?from=membership

## 二、退款与开票

### 1. 如何退款/退订？

退订路径：【首页】→【设置页】→【订阅】→【取消订阅】

> 成功退订后，自动续费服务到期后不再续订，不影响当期有效权益。

### 2. 付费服务可以退费吗？

付费服务属于虚拟服务，一经购买开通即视为成功消费，除法律法规另有规定外，不可退费或转让。

### 3. 如何开具发票？

自助开票：APP/网页 →【我的账户】→【管理-订阅】→【账单】→【开发票】

发票问题：membership@moonshot.cn

## 三、部署方式

### 1. 支持哪些平台？

支持云端部署、Desktop、本地/关联 OpenClaw 等形态。部署方式与能力差异请参考《部署方式分类》及各产品文档（云端/桌面/安卓）。

### 2. 如何创建 Kimi Claw？

1. 登录 https://kimi.com/bot
2. 点击「创建」Kimi Claw
3. 等待系统自动配置（约几分钟）
4. 修改昵称，设置个性化名称
5. 在「设置 → 聊天频道」配置平台

详细指南：https://www.kimi.com/help/kimi-claw/overview
