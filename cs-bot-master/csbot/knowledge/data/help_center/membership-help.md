# 会员权益到账与查询

购买会员后，权益通常会即时生效。如遇到问题，请参考以下指引。

## 会员权益未到账

如果支付完成但会员权益未到账，请按以下步骤排查：

1. **检查支付状态**：确认支付是否实际完成，查看银行或支付平台的扣款记录。部分支付方式可能存在延迟，建议等待几分钟后刷新页面。
> **注意**: **确认账号一致性**：确保你登录的 Kimi 账号与支付时使用的账号是同一个。如果你有多个 Kimi 账号，请切换检查。

2. **尝试重新登录**：退出当前账号后重新登录，权益状态可能会刷新。
3. **联系客服**：如果以上方式均未解决，请发送邮件至 **membership@moonshot.cn**，附上支付截图和账号信息，我们将尽快为你处理。

## 在哪里查看剩余额度？

你可以直接访问[会员订阅页面](https://www.kimi.com/membership/subscription)，点击「我的订阅」进入查询。

**网页版（kimi.com）**：

1. 点击页面左下角头像或「我的」。
2. 进入「设置 → 订阅」页面。
3. 即可查看当前套餐信息和剩余额度。

**手机 App（iOS / Android）**：

1. 打开 Kimi App，点击「设置」标签。
2. 进入「设置 → 会员计划 → 我的订阅」。
3. 页面将展示当前套餐详情及额度使用情况。

# 取消自动续费

如果你不再需要 Kimi 会员服务，可以随时取消自动续费。取消后，当前计费周期内的会员权益仍然有效，到期后将不再续费。

## 如何取消连续自动订阅？

根据你的订阅渠道，选择对应的取消方式：

### 通过 Kimi 页面取消

1. 登录 Kimi（网页版或手机 App）。
2. 进入「设置 → 订阅管理」。
3. 点击「取消自动续费」或「管理订阅」。
4. 确认取消即可。

### 通过微信取消

1. 打开微信，进入「我 → 服务 → 钱包」。
2. 点击右上角「账单」→搜索「月之暗面」→「管理扣费服务」（或「自动续费」）。
3. 点击找到扣费账单。
4. 点击「关闭扣费服务」并确认。

### 通过支付宝取消

1. 打开支付宝，进入「我的 → 设置」。
2. 点击「支付设置 → 免密支付/自动扣款」。
3. 找到「Kimi 会员」，点击进入。
4. 点击「关闭服务」并确认。

### 通过 Apple ID 取消（iOS 用户）

1. 打开 iPhone「设置」。
2. 点击顶部你的 Apple ID 头像。
3. 进入「订阅」。
4. 找到「Kimi」订阅，点击「取消订阅」。

### 通过 Mac 取消

1. 打开 Mac 上的 App Store。
2. 点击左下角你的头像 → 「账户设置」。
3. 在「订阅」栏找到「Kimi」，点击「管理」。
4. 选择「取消订阅」并确认。

> **提示**: 取消自动续费后，当前订阅周期内的权益将保留至到期日。如需重新订阅，随时可在 Kimi 订阅页面操作。

# Kimi会员自助开票

Kimi 支持用户自助开具电子发票，操作简单便捷。你可以直接访问[会员订阅页面](https://www.kimi.com/membership/subscription)进入开票流程。

## 开票操作步骤

1. 登录 Kimi，进入「设置 → 管理 → 订阅」页面。
2. 找到需要开票的订单，点击「开发票」。
3. 选择发票类型并填写开票信息。
4. 确认信息无误后提交申请。
5. 发票将以电子版形式发送到你填写的邮箱。

## 支持的发票类型

| 发票类型 | 说明 |
| --- | --- |
| 普通个人发票 | 抬头为个人姓名，无需税号 |
| 普通企业发票 | 需填写企业名称和纳税人识别号 |
| 增值税专用发票 | 需填写企业完整开票信息（名称、税号、地址、电话、开户行及账号） |

## 注意事项

> **信息**: - 电子发票与纸质发票具有同等法律效力。
- 发票金额为实际支付金额（不含优惠券等抵扣部分）。
- 发票通常在提交申请后 1-3 个工作日内开具完成。

## 已开票如何重新发送？

如果发票邮件丢失或需要发送到其他邮箱：

1. 进入「设置 → 管理 → 订阅」页面。
2. 找到已开票的订单，点击「查看发票」或「重新发送」。
3. 输入新的接收邮箱地址，确认发送即可。

如遇开票问题，请联系 membership@moonshot.cn。

## 计费方式

- **按月刷新**：额度在每个计费周期自动刷新。
- **频次限制** 5小时及周频控请以页面提示为准。
- **使用优先级**：系统优先消耗获赠额度，再消耗套餐额度。

> **信息**: **什么是 Token？**

Token 是大语言模型处理文本的最小单位。中文里，1 个汉字通常约等于 1–2 个 token；英文里，1 个单词通常约等于 1–1.5 个 token。你输入的内容和 Kimi 输出的内容都会消耗 token。

**如何估算一个任务的 Token 消耗？**

- 一次普通对话（几百字问答）：约几百到一千 token
- 生成一份简单的 PPT：约消耗 1–2% 月度额度
- 一次深度研究：约消耗 5–10% 月度额度
- 编写一段代码：约消耗 0.5–2% 月度额度

任务越复杂、上下文越长，消耗的 token 越多。你可以在「设置 → 订阅」页面随时查看剩余额度。

# 会员支付相关问题

以下是会员订阅过程中常见的支付相关问题及解决方案。

## 签约成功为什么还不是会员？

如果你完成了支付签约（如微信免密支付、支付宝自动扣款），但会员状态未生效，可能原因如下：

> **警告**: **签约 ≠ 扣款**：签约仅代表你授权了自动扣款，但首次扣费可能尚未完成。请检查银行或支付平台是否实际完成扣款。

- **支付延迟**：部分支付渠道可能有延迟，请稍等几分钟后刷新页面查看。
- **账号不一致**：请确认当前登录账号与签约时使用的账号一致。

如果确认已扣款但权益未到账，请联系 membership@moonshot.cn。

## 支付失败怎么办？

支付失败的常见原因及解决方法：

- **余额不足**：请确保支付账户中有足够余额。
- **银行卡限额**：部分银行卡有单笔或每日限额，请联系银行确认或更换支付方式。
- **网络问题**：请检查网络连接后重试。
- **支付渠道维护**：偶尔支付渠道可能在维护中，请稍后再试。
- **风控拦截**：支付平台可能因安全原因拦截交易，请联系对应支付平台客服。

如果多次尝试仍然失败，请发送邮件至 membership@moonshot.cn 并附上错误截图。

## 被错误自动扣款怎么办？

> **注意**: 如果你发现账户被意外扣款，请按以下步骤排查：首先检查是否有未取消的自动续费，其次确认是否有试用期转正扣费。

1. **检查订阅状态**：登录 Kimi 查看是否存在有效订阅，可能是之前开通的自动续费未取消。
2. **检查试用转正**：如果之前开通了 7 天试用且未在试用期内取消，系统会在试用结束后自动扣费。
3. **查看扣费服务**：在微信/支付宝/Apple ID 中检查是否有 Kimi 的自动扣费授权。
4. **取消续费**：如不需要继续使用，请参考「取消自动续费」文档关闭自动扣费。
5. **联系客服**：如确认非本人操作，请立即联系 membership@moonshot.cn，我们将协助核实并处理。

## 付费服务可以退费吗？

付费服务属于虚拟服务，一经成功购买并开通后即视为成功消费，除法律法规另有规定外，不可退费或转让。请在购买前务必核对服务及权益信息、价格、使用期限，并确认购买账号信息无误。

> **提示**: 如对扣费有异议，请联系 membership@moonshot.cn，我们会协助核实。

## 未成年人消费提醒

未满 18 周岁的未成年人应在监护人指导下阅读相关协议，经监护人同意后方可购买付费服务。监护人应妥善保管账号、设备及密码，避免未成年人未经同意进行付费操作。如因未成年人未经授权付费产生的问题，请联系 membership@moonshot.cn。

# 会员额度更新与使用规则

Kimi 会员采用统一额度池计费，所有会员功能共享同一额度，按实际 token 消耗扣除。

## 什么是统一额度？

我们将原本分散在各功能中的权益整合为一个统一的额度池。你可以在所有会员功能中自由分配使用，包括：

- **Agent 系列**：网站生成、文档处理、PPT 制作、表格处理、深度研究等
- **Kimi Code**：智能编程助手
- **Kimi Claw**：可一键部署的专属助理
- **Kimi Chat**：使用最新模型进行对话
- **其他会员权益**：图片生成等

## 额度如何计算？

额度消耗主要取决于处理任务的 token 消耗量。当你输入或输出更长内容的任务时，会基于 token 用量消耗额度。

> **信息**: 以免费用户为例：
- 生成一份简单的 PPT 可能消耗 1-2% 额度
- 进行一次深度研究可能消耗 5-10% 额度
- 编写一段代码可能消耗 0.5-2% 额度

## 额度刷新规则

- 月度和年度会员的额度均**按月刷新**
- 当期未使用的额度到期后自动清零，不结转、不折现
- 单次对话或任务可能存在 5 小时及 7 天的并发限制，具体以页面提示为准

## 额度使用优先级

> **提示**: 当存在多种额度时，按以下顺序扣除：
1. **优先使用**：获赠额度（如试用赠送、活动奖励）
2. **其次使用**：会员套餐额度

## 额度耗尽处理

> **警告**: - 当前正在进行的任务可以正常完成
- 新任务将提示额度不足，你可以：
  - 等待额度按月自动刷新
  - 升级至更高等级会员

### 会员额度是如何更新的？

1. **刷新节奏**：会员额度按「订阅周期」按月刷新，与自然月无关
2. **不累积**：当期未用完的额度在周期结束后清零，重新发放满额额度
3. **举例**：小明在 12 月 1 日 15:00 开通 Andante（¥49/月），则每月此时刷新额度

# 会员套餐升级与降级

你可以随时根据需要调整 Kimi 会员套餐。以下是升级和降级的操作说明。

## 升级如何计算差额？

1. 升级套餐时，用户需要缴纳升级套餐完整月费。
2. 系统将按天折算当前套餐的剩余价值，进行折算退款。

> **信息**: - 计算公式：**退差价退费 = 旧套餐月费 − 原套餐已使用天数金额折算**
- 升级即时生效，新套餐的额度立即可用。
- 原套餐已使用的额度不影响新套餐额度（新套餐额度重新计算）。

## 订阅错了想升降级？

**升级套餐**：
1. 进入「设置 → 订阅」页面。
2. 选择更高档的套餐，点击「升级」。
3. 系统自动计算差额并完成支付，升级即时生效。

**降级套餐**：

> **警告**: 降级无法直接操作，需要先取消当前订阅，等到当前计费周期结束后，再重新订阅较低档位的套餐。

具体步骤：

1. 取消当前自动续费（参考「取消自动续费」文档）。
2. 当前订阅到期后，会员权益自动失效。
3. 前往订阅页面选择新的较低档位套餐完成订阅。

## 无法升级怎么办？

如果你在 Kimi 中无法找到升级入口或操作失败，可能是因为：

> **注意**: **在其他渠道订阅**：如果你是通过 Apple App Store 或其他第三方渠道订阅的，需要在对应渠道管理订阅。

- **支付方式问题**：请检查绑定的支付方式是否可用。
- **系统异常**：请尝试退出重新登录，或联系 membership@moonshot.cn 获取帮助。
