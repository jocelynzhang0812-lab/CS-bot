# kb_06 — 会员权益

## 一、会员与额度

### 1. 会员套餐有哪些？

| 套餐 | 价格 | Kimi Claw | 额度 |
|------|------|-----------|------|
| Adagio | ¥0/月 | ❌ | 基础额度 |
| Andante | ¥49/月 | ❌ | 30 个 Agent |
| Moderato | ¥99/月 | ❌ | 60 个 Agent |
| Allegretto | ¥199/月 | ✅ | 150 个 Agent |
| Allegro | ¥699/月 | ✅ | 360 个 Agent |

> **注意**：Kimi Claw 功能仅 Allegretto 及以上套餐支持。

### 2. 如何查看额度？

- **Web 端**：左下角头像 → 设置 → 订阅
- **APP 端**：我的 → 会员计划 → 订阅

### 3. 额度用完了怎么办？

- 等待自动刷新（5 小时/周度/月度）
- 升级至更高等级会员
- 参与官方活动获取赠送额度

### 4. API rate limit 报错是什么意思？

表示 API 调用频率或额度已达上限。请检查：
1. 会员额度是否充足
2. 是否触发频次限制（5 小时及周频控）

详细查看：https://www.kimi.com/code/console?from=membership

## 二、退款与开票

### 1. 如何退款/退订？

退订路径：【首页】→【设置页】→【订阅】→【取消订阅】

> 成功退订后，自动续费服务到期后不再续订，不影响当期有效权益。

### 2. 付费服务可以退费吗？

付费服务属于虚拟服务，一经购买开通即视为成功消费，除法律法规另有规定外，不可退费或转让。

### 3. 如何开具发票？

自助开票：APP/网页 →【我的账户】→【管理-订阅】→【账单】→【开发票】

发票问题：membership@moonshot.cn

## 三、退款服务流程（补充）

> **核心原则**：原则上大部分场景不支持用户退款。用户坚持退款的，转接专业客服人员处理。

### 退款原因确认

必须收集：
1. 详细退款原因
2. 任务对话/代码截图（如不属于客服自主处理场景）

如用户未提及退款原因，询问："用户您好，非常抱歉给您带来了不好的体验，方便了解一下您需要退款的主要原因吗？"

如退款原因模糊（如"服务没有达到预期"），需进一步询问："请问 Kimi 具体在哪些方面没有达到您的预期呢？"

如退款原因详细但无法直接判断，需用户提供举证：
- 主要使用 Kimi 对话及相关 Agent 功能的用户 → 请提供对话链接
- 主要使用 Kimi Code 的用户 → 请提供页面截图

**如用户拒绝提供详细退款原因及举证，婉拒退款**："用户您好，很抱歉，如果没有更详细的问题描述及举证，我们无法为您定位具体的使用问题，也无法判断是否可为您办理退款。"

**如用户坚持退款，转接专业客服人员处理。**

### 客服自主处理场景

#### 用户主观原因

**不想订阅/订阅错了/手滑支付了/不知道为什么就付了**
- 话术："用户您好，很抱歉目前的情况不包含在可退款的范围内，我们无法为您进行办理。给您带来的不便，敬请谅解。"
- 如用户不接受，按婉拒退款处理。

**订阅至错误账户**
- 订阅至亲友账户：建议先取消自动订阅，与账户持有人沟通使用，到期后使用自己账户订阅。Kimi 暂不支持会员订阅在账户之间换绑。
- 订阅至未知账户：收集支付订单号，提交技术团队进行问题定位确认。

**订阅错套餐**
- 希望购买开放平台 API 但误订阅 Kimi 会员：引导使用 Kimi Code。目前会员套餐中包含 Kimi Code 权益，可在会员界面获取 API key。如用户不认可，说明 Kimi Code 额度总量比以往更多，计费更少。详见：https://www.kimi.com/coding/docs/
- 订阅错套餐等级：需升级可直接在会员界面点击升级，原有套餐按剩余天数折算退费；需降级暂不支持直接更换，可先取消自动续费，到期后订阅所需套餐。

**订阅升级退费**
- 用户问"升级了会员为什么只退回了一部分钱"：订阅升级时，原有套餐按当前订阅期剩余天数折算退费，新套餐从升级当天开始计算新订阅期。

**账户自动扣款/莫名扣费**
- 询问扣费实际发生在哪天。
- 如扣费发生在进线当天：直接登记退费。"您反馈的扣款来自 Kimi 会员自动续费扣款，请您提供注册手机号，我们将为您办理退款。"
- 如扣费发生在进线当天之前：需提供注册手机号，由技术团队确认会员使用情况。如未使用权益则办理退款；如已使用权益则婉拒。

**会员套餐调整不希望继续订阅**
- 如赠送 API 代金券限时活动下线："Kimi 会员的赠送 API 券活动为限时活动，目前活动已结束。了解到您有开发的需要，目前会员套餐中包含 Kimi Code 权益，可在会员界面获取 API key。"

#### 平台服务原因1：内容/功能与宣传不符

**用户预期功能不在会员权益范围内**
- 对话超过 20w 字无法继续任务：目前"对话可超过 20w 字"不在会员权益承诺范围内，无法办理退款。详见：https://www.kimi.com/user/agreement/membershipBenefits?version=v2
- 存在 3 小时内对话量限制：目前"3 小时对话量无上限"不在会员权益承诺范围内，无法办理退款。
- 模型导致承诺可实现非会员权益内容：目前"由 Kimi 生成的承诺"不在会员权益承诺范围内，无法办理退款。
- **如用户坚持退款，按权益使用情况核实处理，必要时转接专业客服人员。**

**会员权益消耗过快/额度已用完**
- OK Computer/深度研究/PPT 助手额度已用完：询问购买的会员档位，解释对应额度。Andante(¥49) 每月 10 次，Moderato(¥99) 每月 20 次，Allegretto(¥199) 每月 40 次，Allegro(¥699) 每月 100 次。功能根据对话次数进行额度消耗。详见：https://www.kimi.com/user/agreement/membershipBenefits?version=v2
  - 如用户不满：¥49/¥99/¥199 档位引导升级；¥699 档位建议等待下个订阅期刷新。
- Kimi Code 额度已用完：每周额度以会员展示页为准。详见：https://www.kimi.com/membership/pricing
  - 如用户不满：¥49/¥99/¥199 档位引导升级；¥699 档位建议等待下个周期刷新。

**用户认为功能不生效但功能实际正常运行**
- K2 Turbo 无标识/不生效：K2 Turbo 在使用期间保持生效，仅在页面展示上做了调整，可在 Kimi 对话框下方的模型选择处看到 K2 Turbo 标签。

### 客服反馈处理场景

#### 平台服务原因2：支付/账户问题

**非本人订阅**
- 账户被盗刷：不属于平台受理范围，请用户在支付渠道提起申诉。
  - 微信支付：在微信支付中找到对应订单 → "对订单有疑惑" → "交易非本人支付" → "非本人及亲友支付，资金被盗申赔" → 填写申赔信息并提交。
  - 支付宝支付：找到对应订单 → "对此订单有疑问" → "交易非本人支付" → 填写情况并留下联系电话 → "继续举报"。
  - Apple 支付：进入 reportaproblem.apple.com → 登录账号 → "请求退款" → "其他" → 选择 Kimi 订阅项目 → 描述原因并提交。
- 亲友使用用户账户订阅：暂时不属于处理范围，建议优先与亲友沟通，后续保管好账号、设备及密码。

**封禁账号**
- 用户反馈账号封禁无法正常使用申请退款：根据《Kimi 付费服务协议》约定，用户存在违反法律法规、协议或平台规则的行为导致付费服务中止或终止后，无义务退还已经支付的费用。无法办理退款。

#### 平台服务原因3：会员权益与承诺不符

**权益未到账**
- 先请用户提供头像截图，判断是否是会员。
- 头像旁标识为黑底白字的 Andante/Moderato/Allegratto/Allegro → 已是会员，可使用权益。
- 头像旁无标识或标识为白底黑字的"升级" → 不是会员，询问是否有其他注册账户。如其他账户也不是会员，请提供支付订单详情页面截图，提交内部确认。

**权益次数未到账 / 权益次数/额度/并发数少于平台承诺**
- 按照权益使用核实处理。

#### 平台服务原因4：具体功能问题

**Kimi 对话问题**
- 对话发送不出去/发送显示红圈：一段时间内发送消息达到对话上限，建议 2-3 小时后继续使用，注意对话频次。
- Kimi 回复"Kimi 有点累了，可以晚点再问我一遍"：使用高峰期算力不足，建议过一段时间后继续使用。
- Kimi 瞎编乱造、文献根本不存在、违反专业/客观事实：属于模型幻觉，AI 技术本身的限制无法保证生成内容完全准确。已将反馈提交内部，后续会持续优化。
- 订阅后功能体验不佳、体验下降、体验不及竞品：按照婉拒退款处理。

**PPT 助手问题**
- 生成 PPT 时卡住、一直没法生成 PPT 文件、PPT 无法下载、任务闪退：按照任务失败补偿沟通补充额度。
- 不订阅会员也可以使用【PPT 助手】：按照婉拒退款处理。
- PPT 生成速度慢、会员权益承诺 4 倍速生成但没感觉到：按照婉拒退款处理。
- PPT 里有指令之外的内容、不完全按照指令生成：按照婉拒退款处理。
- PPT 模板少、做出来 PPT 不好看：按照婉拒退款处理。

**OK Computer/深度研究问题**
- 任务一直没有完成、任务进行中断：
  - 深度研究功能：此次任务不扣除用户权益额度，引导用户创建新对话，输入相同指令再次生成结果。
  - OK Computer 功能：此次任务扣除用户权益额度，引导用户去新任务里生成结果，并按照任务失败补偿沟通补充额度。
- 想制作 PPT 但生成了 HTML 文件、承诺发送至网盘/邮箱：按照任务失败补偿沟通补充额度。
- 任务没有生成想要的文件：按照任务失败补偿沟通补充额度。
- 功能使用体验不佳：按照使用体验补偿沟通补充额度。

## 四、部署方式

### 1. 支持哪些平台？

支持云端部署、Desktop、本地/关联 OpenClaw 等形态。部署方式与能力差异请参考《部署方式分类》及各产品文档（云端/桌面/安卓）。

### 2. 如何创建 Kimi Claw？

1. 登录 https://kimi.com/bot
2. 点击「创建」Kimi Claw
3. 等待系统自动配置（约几分钟）
4. 修改昵称，设置个性化名称
5. 在「设置 → 聊天频道」配置平台

详细指南：https://www.kimi.com/help/kimi-claw/overview
