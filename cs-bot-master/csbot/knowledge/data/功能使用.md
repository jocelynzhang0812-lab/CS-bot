## 1. 功能使用通用问题

> 本文为通用能力总览，具体能力以各产品文档（云端/Desktop/Android）为准。

### 1.1 Kimi Claw 可以收发文件吗？

- **发送文件给 Kimi Claw：** 可以直接通过对话发送图片、文件等，Kimi Claw 会使用工具查看
- **从 Kimi Claw 接收文件：** 最新版本已支持，若无法发送请升级插件到最新版本
- **备选方式：** 可使用飞书机器人完成文件获取

### 1.2 无法下载产物文件怎么办？

Web 端（https://www.kimi.com/bot）已上线文件查看功能：

点击页面右上角【工作空间】入口（📂 图标），即可浏览和下载所需文件。

> iOS / Android 端相关功能正在开发中。

### 1.3 可以使用命令行（Terminal）吗？

请按产品类型区分：

- **云端 Kimi Claw：** 支持
  1) 设置 → 打开终端  
  2) 在对话中让 Bot 执行命令
- **Kimi Claw Desktop：** 不支持独立 Terminal，仅支持对话框命令（如 `/help`、`/status`）
- **Kimi Claw Android：** 不支持独立 Terminal，仅支持对话交互
- **关联已有 OpenClaw：** 取决于用户自行配置，Kimi 插件不提供 Terminal 功能

> **注意：** 云端 Terminal 和插件共享交互通路，重启 OpenClaw Gateway 会导致 Terminal 断开连接。

### 1.4 如何在关联的 OpenClaw 里卸载 Kimi 插件？

在关联 OpenClaw 的设备上运行：

```bash
bash <(curl -fsSL https://cdn.kimi.com/kimi-claw/uninstall.sh)
```

> **警告：** 此命令仅适用于关联的 OpenClaw 设备。**切勿在一键部署的 Kimi Claw 设备上运行**，否则将永久断开与 Kimi Claw 的连接且无法修复。

### 1.5 能否手动升级 OpenClaw 版本？

技术上可以实现，但**不推荐自行升级**。升级过程存在兼容性和稳定性风险。建议：

- 等待官方提供升级途径
- 如需升级，使用"恢复初始版本"方式

### 1.6 升级 OpenClaw 版本后不能用了怎么办？

当前原因：OpenClaw 版本升级与插件版本存在兼容性问题。

### 1.7 鸿蒙系统支持吗？

Kimi Claw 现已支持 HarmonyOS。

### 1.8 如何修改 Bot 人设？

直接对话告诉它：
- "从现在开始你叫 XX，身份是 XX"
- "每次回复开头先说 XX"
- "用 XX 风格回复"

### 1.9 如何查看/导出记忆？

```
/memory                # 查看记忆条目
/memory search <关键词> # 搜索记忆
/memory export         # 导出记忆备份
```

### 1.10 如何新建 Session？

发送 `/new` 创建新 Session，保留历史对话。

### 1.11 Kimi Claw 里的 kimi-search 插件可以给我的其他 OpenClaw 用吗？

目前我们没有开放 **kimi-search** 插件的公开使用，不支持跨 `OpenClaw` 实例共享或迁移。如需在多个 OpenClaw 中使用搜索能力，建议参考配置中的 **Search API** 自行接入。
