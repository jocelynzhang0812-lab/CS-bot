# kb_05 — 问题上报与人工客服处理

### 1.1 如何上报问题？

如遇问题需要人工客服协助，请提供以下信息，我们会记录并推动给人工处理：

- **Kimi Claw Bot ID**（没有 Bot ID 无法处理）
- 产品类型：云端 / Desktop / Android / 群聊
- 问题截图和过程描述

> 您的问题已记录，我们会尽快推动给人工客服处理，请耐心等待。

### 1.3 反馈模板（建议直接复制）

```text
1) Bot ID：
2) 部署方式：云端 / Desktop / Android / 群聊
3) 问题描述：
4) 复现步骤：
5) 截图/日志：
```

### 1.2 如何获取 Bot ID？

1. 在 Kimi 侧边栏点击该 bot 的头像
2. 右侧设置面板中，昵称下方直接显示 ID（如 `19xxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`）
3. 点击复制即可
