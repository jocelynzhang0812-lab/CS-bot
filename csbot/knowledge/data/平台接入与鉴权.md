# kb_01 — 平台接入与鉴权

> 适用范围：以云端 Kimi Claw 接入与鉴权为主；Desktop/Android 仅适用相同渠道原则。

## 1. 平台支持与接入

### 1.1 Kimi Claw 支持哪些平台？

Kimi Claw 目前在以下平台可以使用：

- 网页端：kimi.com
- 手机端：iOS / Android

### 1.2 为什么我的 Kimi Claw 里看不到"一键扫描关联微信 / 飞书"？

**原因：** 企微 / 微信插件依赖新版本 OpenClaw。若 Kimi Claw 是在 **3 月 22 日之前**创建的，暂不支持该功能。

**解决方案（二选一）：**

| 方案 | 适用场景 | 操作方式 |
|------|----------|----------|
| 方案一：保留文件升级 | 有文件需要保留 | 设置 → 恢复初始版本（保留工作空间，需重新配置聊天软件） |
| 方案二：删除重建 | 无文件需要保留 | 删除 Bot，重新新建 |

**无损升级方式：** 等待下一个版本更新，届时老版本 OpenClaw 将自动升级至支持微信 / 飞书最新插件的版本，请关注 Kimi Claw 页面内的升级提示。

## 2. 飞书接入与配置

### 2.1 在飞书群里 @ 机器人没有反应怎么办？

**原因：** 飞书机器人默认策略不响应群内对话。

**解决方法一（自然语言）：**

发送以下内容给 Kimi Claw：

```
将飞书的群聊策略调整为 open
```

然后重启 Kimi Claw。

**解决方法二（直接配置）：**

发送以下配置给机器人：

```
你帮我加上这几个配置：
dmPolicy: pairing（私聊需要配对）
groupPolicy: open（所有群都响应）
requireMention: true（群里需要 @ 才响应）
```

配置后重启 Kimi Claw 即可生效。

### 2.2 升级 Kimi Claw 版本后飞书聊不了？

**原因：** 升级会安装最新飞书插件，可能需要重新鉴权。

**判断依据：** 若收到以下内容：

```
OpenClaw: access not configured.
ou_xxxxxxx
Pairing code: xxxxxxxx
Ask the bot owner to approve with:
openclaw pairing approve feishu xxxxxxxx
```

说明飞书需要重新鉴权。

**解决方法：**

1. 将上述提示直接发给 Kimi Claw，让它帮你完成配对；
2. 或打开终端，输入 `openclaw pairing approve feishu xxxxxxxx` 完成配对。

若以上方式均无法完成，请联系人工客服进一步处理。

### 2.3 OpenClaw 提示 "access not configured" 怎么办？

**完整报错示例：**

```
Pairing code: xxxxxxx
Ask the bot owner to approve with:
openclaw pairing approve feishu xxxxx
```

**原因：** 飞书机器人需要 Kimi Claw 批准访问。

**解决方法：** 提示中已包含飞书账号信息，直接在网页端或 App 里将这段话发给 Kimi Claw，飞书会自动完成授权。

### 2.4 如何关掉飞书 / 企业微信 / 微博里显示的 Reasoning 部分？

发送命令：

```
/reasoning off
```

> 说明：此操作隐藏思考过程的展示，并不关闭模型内部的思考。可用 `/reasoning on` 或 `/reasoning stream` 重新打开显示。

### 2.5 飞书 Token / 认证相关场景

| 场景 | 用户看到的报错 | 报错类型 |
|------|---------------|----------|
| 飞书 Token 失效 | "飞书连接失败" | `failed to obtain token` |
| WebSocket 认证失败 | "无法接收飞书消息" | `code: 1000040345, system busy` |
| Axios 400 错误 | 飞书功能不可用 | `AxiosError: Request failed with status code 400` |

**解决方法：** 

1. 将上述提示直接发给 Kimi Claw，让它帮你完成配对；
2. 或打开终端，输入 `openclaw pairing approve feishu xxxxxxxx` 完成配对。

## 3. 微信 / 企业微信接入

### 3.1 为什么微信消息没有同步到 Kimi？

这是 OpenClaw 的官方设定。微信消息属于微信渠道频道，聊天记录只在微信里可以看到。同理，飞书 / 企微等其他 IM 的信息也不会在 Kimi 中显示。
