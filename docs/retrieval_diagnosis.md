# 召回质量诊断报告

## 诊断方法

-  Golden Queries：`tests/queries_golden.txt`（20 条真实用户问题）
-  诊断脚本：`scripts/diagnose_retrieval.py`
-  评判标准：
  - `[OK]` chunk 内容与问题相关
  - `[WRONG]` chunk 内容无关（向量/切割问题）
  - `[MISSING]` 相关内容存在于知识库但没被召回（用词差异问题）
  - `[EMPTY]` 召回为空（阈值或索引问题）

## 诊断结果汇总

|------|------|------|------|
| `[OK]` | - | - | 召回正确 |
| `[WRONG]` | - | - | 召回不相关 |
| `[MISSING]` | - | - | 相关文档未召回 |
| `[EMPTY]` | - | - | 召回为空 |
